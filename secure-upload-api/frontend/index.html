Security Pipeline
01 — File Validation
Size limits enforced. Extension block list checked. MIME type read from magic bytes — not filename. Polyglot files detected.
02 — ClamAV Scan
File is streamed through clamd. Malware, trojans, and known threats are detected. Infected files are rejected immediately.
03 — Encryption
Fernet symmetric encryption (AES-128-CBC + HMAC-SHA256). Keys derived via PBKDF2 with 600,000 iterations.
04 — Integrity
SHA-256 checksum recorded at upload. Verified on every download. Tampered or corrupted files are refused.
